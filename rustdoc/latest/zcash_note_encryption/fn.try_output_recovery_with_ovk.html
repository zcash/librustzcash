Expand description

Recovery of the full note plaintext by the sender.

Attempts to decrypt and validate the given enc_ciphertext using the given ovk. If successful, the corresponding note and memo are returned, along with the address to which the note was sent.

Implements Zcash Protocol Specification section 4.19.3.

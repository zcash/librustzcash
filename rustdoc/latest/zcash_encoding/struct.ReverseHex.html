Struct ReverseHex 

Source 
pub struct ReverseHex;
Expand description

This handles the legacy format from Bitcoin Core’s uint256::GetHex, which reverses bytes before hex encoding. zcashd applied this to all 32-byte JSON-RPC values (transaction IDs, block hashes, Sprout field elements, etc.), though non-32-byte hex values are not reversed.

pub fn decode(hex_str: &str) -> Option<[u8; 32]>

Decodes a reversed-hex string into a byte array by decoding the hex and reversing the order of the bytes.

Returns None if the hex decoding fails or if the output is the wrong size.

A ZIP 48 P2SH multisig full viewing key.

This provides the necessary derivation capability to view all funds controlled by a P2SH multisig account.

pub fn standard( threshold: u8, key_info: Vec<AccountPubKey>, ) -> Result<Self, FullViewingKeyError>

Constructs a full viewing key for a standard ZIP 48 account.

This uses the standard wallet descriptor template. For example:

  • threshold == 2
  • key_info.len() == 3
  • Wallet descriptor template: "sh(sortedmulti(2,@0/**,@1/**,@2/**))"

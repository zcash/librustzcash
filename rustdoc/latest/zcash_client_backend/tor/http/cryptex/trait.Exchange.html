An exchange that can be queried for ZEC data.

Required Methods§

Source

fn query_zec_to_usd( &self, client: &Client, ) -> impl Future<Output = Result<ExchangeData, Error>> + Send

Queries data about the USD/ZEC pair.

The returned bid and ask data must be denominated in USD, i.e. the latest bid and ask for 1 ZEC.

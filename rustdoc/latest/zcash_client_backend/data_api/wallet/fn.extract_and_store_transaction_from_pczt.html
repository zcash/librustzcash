Expand description

Finalizes the given PCZT, and persists the transaction to the wallet database.

The PCZT should have been created via create_pczt_from_proposal, which adds metadata necessary for the wallet backend.

Returns the transaction ID for the resulting transaction.

  • sapling_vk is optional to allow the caller to check whether a PCZT has Sapling with pczt::roles::prover::Prover::requires_sapling_proofs, and avoid downloading the Sapling parameters if they are not needed. If sapling_vk is None, and the PCZT has a Sapling bundle, this function will return an error.
  • orchard_vk is optional to allow the caller to control where the Orchard verifying key is generated or cached. If orchard_vk is None, and the PCZT has an Orchard bundle, an Orchard verifying key will be generated on the fly.

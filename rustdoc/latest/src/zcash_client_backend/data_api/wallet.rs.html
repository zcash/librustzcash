1//! # Functions for creating Zcash transactions that spend funds belonging to the wallet
2//!
3//! This module contains several different ways of creating Zcash transactions. This module is
4//! designed around the idea that a Zcash wallet holds its funds in notes in either the Orchard
5//! or Sapling shielded pool. In order to better preserve users' privacy, it does not provide any
6//! functionality that allows users to directly spend transparent funds except by sending them to a
7//! shielded internal address belonging to their wallet.
8//!
9//! The important high-level operations provided by this module are [`propose_transfer`],
10//! and [`create_proposed_transactions`].
11//!
12//! [`propose_transfer`] takes a [`TransactionRequest`] object, selects inputs notes and
13//! computes the fees required to satisfy that request, and returns a [`Proposal`] object that
14//! describes the transaction to be made.
15//!
16//! [`create_proposed_transactions`] constructs one or more Zcash [`Transaction`]s based upon a
17//! provided [`Proposal`], stores them to the wallet database, and returns the [`TxId`] for each
18//! constructed transaction to the caller. The caller can then use the
19//! [`WalletRead::get_transaction`] method to retrieve the newly constructed transactions. It is
20//! the responsibility of the caller to retrieve and serialize the transactions and submit them for
21//! inclusion into the Zcash blockchain.
22//!
23#![cfg_attr(
24    feature = "transparent-inputs",
25    doc = "
26Another important high-level operation provided by this module is [`propose_shielding`], which
27takes a set of transparent source addresses, and constructs a [`Proposal`] to send those funds
28to a wallet-internal shielded address, as described in [ZIP 316](https://zips.z.cash/zip-0316).
29
30[`propose_shielding`]: crate::data_api::wallet::propose_shielding
31"
32)]

170/// Scans a [`Transaction`] for any information that can be decrypted by the accounts in
171/// the wallet, and saves it to the wallet.
172pub fn decrypt_and_store_transaction<ParamsT, DbT>(
173    params: &ParamsT,
174    data: &mut DbT,
175    tx: &Transaction,
176    mined_height: Option<BlockHeight>,
177) -> Result<(), DbT::Error>
178where
179    ParamsT: consensus::Parameters,
180    DbT: WalletWrite,
181{
182    // Fetch the UnifiedFullViewingKeys we are tracking
183    let ufvks = data.get_unified_full_viewing_keys()?;
184
185    // Height is block height for mined transactions, and the "mempool height" (chain height + 1)
186    // for mempool transactions.
187    let height = mined_height.map(Ok).unwrap_or_else(|| {
188        Ok(data
189            .get_tx_height(tx.txid())?
190            .or(data.chain_height()?.map(|max_height| max_height + 1))
191            .or_else(|| params.activation_height(NetworkUpgrade::Sapling))
192            .expect("Sapling activation height must be known."))
193    })?;
194
195    data.store_decrypted_tx(decrypt_transaction(params, height, tx, &ufvks))?;
196
197    Ok(())
198}

265/// Select transaction inputs, compute fees, and construct a proposal for a transaction or series
266/// of transactions that can then be authorized and made ready for submission to the network with
267/// [`create_proposed_transactions`].
268#[allow(clippy::too_many_arguments)]
269#[allow(clippy::type_complexity)]
270pub fn propose_transfer<DbT, ParamsT, InputsT, ChangeT, CommitmentTreeErrT>(
271    wallet_db: &mut DbT,
272    params: &ParamsT,
273    spend_from_account: <DbT as InputSource>::AccountId,
274    input_selector: &InputsT,
275    change_strategy: &ChangeT,
276    request: zip321::TransactionRequest,
277    min_confirmations: NonZeroU32,
278) -> Result<
279    Proposal<ChangeT::FeeRule, <DbT as InputSource>::NoteRef>,
280    ProposeTransferErrT<DbT, CommitmentTreeErrT, InputsT, ChangeT>,
281>
282where
283    DbT: WalletRead + InputSource<Error = <DbT as WalletRead>::Error>,
284    <DbT as InputSource>::NoteRef: Copy + Eq + Ord,
285    ParamsT: consensus::Parameters + Clone,
286    InputsT: InputSelector<InputSource = DbT>,
287    ChangeT: ChangeStrategy<MetaSource = DbT>,
288{
289    let (target_height, anchor_height) = wallet_db
290        .get_target_and_anchor_heights(min_confirmations)
291        .map_err(|e| Error::from(InputSelectorError::DataSource(e)))?
292        .ok_or_else(|| Error::from(InputSelectorError::SyncRequired))?;
293
294    input_selector
295        .propose_transaction(
296            params,
297            wallet_db,
298            target_height,
299            anchor_height,
300            spend_from_account,
301            request,
302            change_strategy,
303        )
304        .map_err(Error::from)
305}

307/// Proposes making a payment to the specified address from the given account.
308///
309/// Returns the proposal, which may then be executed using [`create_proposed_transactions`].
310/// Depending upon the recipient address, more than one transaction may be constructed
311/// in the execution of the returned proposal.
312///
313/// This method uses the basic [`GreedyInputSelector`] for input selection.
314///
315/// Parameters:
316/// * `wallet_db`: A read/write reference to the wallet database.
317/// * `params`: Consensus parameters.
318/// * `fee_rule`: The fee rule to use in creating the transaction.
319/// * `spend_from_account`: The unified account that controls the funds that will be spent
320///   in the resulting transaction. This procedure will return an error if the
321///   account ID does not correspond to an account known to the wallet.
322/// * `min_confirmations`: The minimum number of confirmations that a previously
323///   received note must have in the blockchain in order to be considered for being
324///   spent. A value of 10 confirmations is recommended and 0-conf transactions are
325///   not supported.
326/// * `to`: The address to which `amount` will be paid.
327/// * `amount`: The amount to send.
328/// * `memo`: A memo to be included in the output to the recipient.
329/// * `change_memo`: A memo to be included in any change output that is created.
330/// * `fallback_change_pool`: The shielded pool to which change should be sent if
331///   automatic change pool determination fails.
332#[allow(clippy::too_many_arguments)]
333#[allow(clippy::type_complexity)]
334pub fn propose_standard_transfer_to_address<DbT, ParamsT, CommitmentTreeErrT>(
335    wallet_db: &mut DbT,
336    params: &ParamsT,
337    fee_rule: StandardFeeRule,
338    spend_from_account: <DbT as InputSource>::AccountId,
339    min_confirmations: NonZeroU32,
340    to: &Address,
341    amount: Zatoshis,
342    memo: Option<MemoBytes>,
343    change_memo: Option<MemoBytes>,
344    fallback_change_pool: ShieldedProtocol,
345) -> Result<
346    Proposal<StandardFeeRule, DbT::NoteRef>,
347    ProposeTransferErrT<
348        DbT,
349        CommitmentTreeErrT,
350        GreedyInputSelector<DbT>,
351        SingleOutputChangeStrategy<DbT>,
352    >,
353>
354where
355    ParamsT: consensus::Parameters + Clone,
356    DbT: InputSource,
357    DbT: WalletRead<
358        Error = <DbT as InputSource>::Error,
359        AccountId = <DbT as InputSource>::AccountId,
360    >,
361    DbT::NoteRef: Copy + Eq + Ord,
362{
363    let request = zip321::TransactionRequest::new(vec![Payment::new(
364        to.to_zcash_address(params),
365        amount,
366        memo,
367        None,
368        None,
369        vec![],
370    )
371    .ok_or(Error::MemoForbidden)?])
372    .expect(
373        "It should not be possible for this to violate ZIP 321 request construction invariants.",
374    );
375
376    let input_selector = GreedyInputSelector::<DbT>::new();
377    let change_strategy = SingleOutputChangeStrategy::<DbT>::new(
378        fee_rule,
379        change_memo,
380        fallback_change_pool,
381        DustOutputPolicy::default(),
382    );
383
384    propose_transfer(
385        wallet_db,
386        params,
387        spend_from_account,
388        &input_selector,
389        &change_strategy,
390        request,
391        min_confirmations,
392    )
393}

446/// Construct, prove, and sign a transaction or series of transactions using the inputs supplied by
447/// the given proposal, and persist it to the wallet database.
448///
449/// Returns the database identifier for each newly constructed transaction, or an error if
450/// an error occurs in transaction construction, proving, or signing.
451///
452/// When evaluating multi-step proposals, only transparent outputs of any given step may be spent
453/// in later steps; attempting to spend a shielded note (including change) output by an earlier
454/// step is not supported, because the ultimate positions of those notes in the global note
455/// commitment tree cannot be known until the transaction that produces those notes is mined,
456/// and therefore the required spend proofs for such notes cannot be constructed.
457#[allow(clippy::too_many_arguments)]
458#[allow(clippy::type_complexity)]
459pub fn create_proposed_transactions<DbT, ParamsT, InputsErrT, FeeRuleT, ChangeErrT, N>(
460    wallet_db: &mut DbT,
461    params: &ParamsT,
462    spend_prover: &impl SpendProver,
463    output_prover: &impl OutputProver,
464    usk: &UnifiedSpendingKey,
465    ovk_policy: OvkPolicy,
466    proposal: &Proposal<FeeRuleT, N>,
467) -> Result<NonEmpty<TxId>, CreateErrT<DbT, InputsErrT, FeeRuleT, ChangeErrT, N>>
468where
469    DbT: WalletWrite + WalletCommitmentTrees,
470    ParamsT: consensus::Parameters + Clone,
471    FeeRuleT: FeeRule,
472{
473    // The set of transparent `StepOutput`s available and unused from prior steps.
474    // When a transparent `StepOutput` is created, it is added to the map. When it
475    // is consumed, it is removed from the map.
476    #[cfg(feature = "transparent-inputs")]
477    let mut unused_transparent_outputs = HashMap::new();
478
479    let account_id = wallet_db
480        .get_account_for_ufvk(&usk.to_unified_full_viewing_key())
481        .map_err(Error::DataSource)?
482        .ok_or(Error::KeyNotRecognized)?
483        .id();
484
485    let mut step_results = Vec::with_capacity(proposal.steps().len());
486    for step in proposal.steps() {
487        let step_result: StepResult<_> = create_proposed_transaction(
488            wallet_db,
489            params,
490            spend_prover,
491            output_prover,
492            usk,
493            account_id,
494            ovk_policy.clone(),
495            proposal.fee_rule(),
496            proposal.min_target_height(),
497            &step_results,
498            step,
499            #[cfg(feature = "transparent-inputs")]
500            &mut unused_transparent_outputs,
501        )?;
502        step_results.push((step, step_result));
503    }
504
505    // Ephemeral outputs must be referenced exactly once.
506    #[cfg(feature = "transparent-inputs")]
507    for so in unused_transparent_outputs.into_keys() {
508        if let StepOutputIndex::Change(i) = so.output_index() {
509            // references have already been checked
510            if step_results[so.step_index()].0.balance().proposed_change()[i].is_ephemeral() {
511                return Err(ProposalError::EphemeralOutputLeftUnspent(so).into());
512            }
513        }
514    }
515
516    let created = time::OffsetDateTime::now_utc();
517
518    // Store the transactions only after creating all of them. This avoids undesired
519    // retransmissions in case a transaction is stored and the creation of a subsequent
520    // transaction fails.
521    let mut transactions = Vec::with_capacity(step_results.len());
522    let mut txids = Vec::with_capacity(step_results.len());
523    #[allow(unused_variables)]
524    for (_, step_result) in step_results.iter() {
525        let tx = step_result.build_result.transaction();
526        transactions.push(SentTransaction::new(
527            tx,
528            created,
529            proposal.min_target_height(),
530            account_id,
531            &step_result.outputs,
532            step_result.fee_amount,
533            #[cfg(feature = "transparent-inputs")]
534            &step_result.utxos_spent,
535        ));
536        txids.push(tx.txid());
537    }
538
539    wallet_db
540        .store_transactions_to_be_sent(&transactions)
541        .map_err(Error::DataSource)?;
542
543    Ok(NonEmpty::from_vec(txids).expect("proposal.steps is NonEmpty"))
544}

644    prior_step_results: &[(&Step<N>, StepResult<<DbT as WalletRead>::AccountId>)],
645    proposal_step: &Step<N>,
646    #[cfg(feature = "transparent-inputs")] unused_transparent_outputs: &mut HashMap<
647        StepOutput,
648        (TransparentAddress, OutPoint),
649    >,
650) -> Result<
651    BuildState<'static, ParamsT, DbT::AccountId>,
652    CreateErrT<DbT, InputsErrT, FeeRuleT, ChangeErrT, N>,
653>
654where
655    DbT: WalletWrite + WalletCommitmentTrees,
656    ParamsT: consensus::Parameters + Clone,
657    FeeRuleT: FeeRule,
658{
659    #[cfg(feature = "transparent-inputs")]
660    let step_index = prior_step_results.len();
661
662    // We only support spending transparent payments or transparent ephemeral outputs from a
663    // prior step (when "transparent-inputs" is enabled).
664    //
665    // TODO: Maybe support spending prior shielded outputs at some point? Doing so would require
666    // a higher-level approach in the wallet that waits for transactions with shielded outputs to
667    // be mined and only then attempts to perform the next step.
668    #[allow(clippy::never_loop)]
669    for input_ref in proposal_step.prior_step_inputs() {
670        let (prior_step, _) = prior_step_results
671            .get(input_ref.step_index())
672            .ok_or(ProposalError::ReferenceError(*input_ref))?;
673
674        #[allow(unused_variables)]
675        let output_pool = match input_ref.output_index() {
676            StepOutputIndex::Payment(i) => prior_step.payment_pools().get(&i).cloned(),
677            StepOutputIndex::Change(i) => match prior_step.balance().proposed_change().get(i) {
678                Some(change) if !change.is_ephemeral() => {
679                    return Err(ProposalError::SpendsChange(*input_ref).into());
680                }
681                other => other.map(|change| change.output_pool()),
682            },
683        }
684        .ok_or(ProposalError::ReferenceError(*input_ref))?;
685
686        // Return an error on trying to spend a prior output that is not supported.
687        #[cfg(feature = "transparent-inputs")]
688        if output_pool != PoolType::TRANSPARENT {
689            return Err(Error::ProposalNotSupported);
690        }
691        #[cfg(not(feature = "transparent-inputs"))]
692        return Err(Error::ProposalNotSupported);
693    }
694
695    let (sapling_anchor, sapling_inputs) = if proposal_step
696        .involves(PoolType::Shielded(ShieldedProtocol::Sapling))
697    {
698        proposal_step.shielded_inputs().map_or_else(
699            || Ok((Some(sapling::Anchor::empty_tree()), vec![])),
700            |inputs| {
701                wallet_db.with_sapling_tree_mut::<_, _, Error<_, _, _, _, _, _>>(|sapling_tree| {
702                    let anchor = sapling_tree
703                        .root_at_checkpoint_id(&inputs.anchor_height())?
704                        .ok_or(ProposalError::AnchorNotFound(inputs.anchor_height()))?
705                        .into();
706
707                    let sapling_inputs = inputs
708                        .notes()
709                        .iter()
710                        .filter_map(|selected| match selected.note() {
711                            Note::Sapling(note) => sapling_tree
712                                .witness_at_checkpoint_id_caching(
713                                    selected.note_commitment_tree_position(),
714                                    &inputs.anchor_height(),
715                                )
716                                .and_then(|witness| {
717                                    witness
718                                        .ok_or(ShardTreeError::Query(QueryError::CheckpointPruned))
719                                })
720                                .map(|merkle_path| {
721                                    Some((selected.spending_key_scope(), note, merkle_path))
722                                })
723                                .map_err(Error::from)
724                                .transpose(),
725                            #[cfg(feature = "orchard")]
726                            Note::Orchard(_) => None,
727                        })
728                        .collect::<Result<Vec<_>, Error<_, _, _, _, _, _>>>()?;
729
730                    Ok((Some(anchor), sapling_inputs))
731                })
732            },
733        )?
734    } else {
735        (None, vec![])
736    };
737
738    #[cfg(feature = "orchard")]
739    let (orchard_anchor, orchard_inputs) = if proposal_step
740        .involves(PoolType::Shielded(ShieldedProtocol::Orchard))
741    {
742        proposal_step.shielded_inputs().map_or_else(
743            || Ok((Some(orchard::Anchor::empty_tree()), vec![])),
744            |inputs| {
745                wallet_db.with_orchard_tree_mut::<_, _, Error<_, _, _, _, _, _>>(|orchard_tree| {
746                    let anchor = orchard_tree
747                        .root_at_checkpoint_id(&inputs.anchor_height())?
748                        .ok_or(ProposalError::AnchorNotFound(inputs.anchor_height()))?
749                        .into();
750
751                    let orchard_inputs = inputs
752                        .notes()
753                        .iter()
754                        .filter_map(|selected| match selected.note() {
755                            #[cfg(feature = "orchard")]
756                            Note::Orchard(note) => orchard_tree
757                                .witness_at_checkpoint_id_caching(
758                                    selected.note_commitment_tree_position(),
759                                    &inputs.anchor_height(),
760                                )
761                                .and_then(|witness| {
762                                    witness
763                                        .ok_or(ShardTreeError::Query(QueryError::CheckpointPruned))
764                                })
765                                .map(|merkle_path| Some((note, merkle_path)))
766                                .map_err(Error::from)
767                                .transpose(),
768                            Note::Sapling(_) => None,
769                        })
770                        .collect::<Result<Vec<_>, Error<_, _, _, _, _, _>>>()?;
771
772                    Ok((Some(anchor), orchard_inputs))
773                })
774            },
775        )?
776    } else {
777        (None, vec![])
778    };
779    #[cfg(not(feature = "orchard"))]
780    let orchard_anchor = None;
781
782    // Create the transaction. The type of the proposal ensures that there
783    // are no possible transparent inputs, so we ignore those here.
784    let mut builder = Builder::new(
785        params.clone(),
786        min_target_height,
787        BuildConfig::Standard {
788            sapling_anchor,
789            orchard_anchor,
790        },
791    );
792
793    #[cfg(all(feature = "transparent-inputs", not(feature = "orchard")))]
794    let has_shielded_inputs = !sapling_inputs.is_empty();
795    #[cfg(all(feature = "transparent-inputs", feature = "orchard"))]
796    let has_shielded_inputs = !(sapling_inputs.is_empty() && orchard_inputs.is_empty());
797
798    for (_sapling_key_scope, sapling_note, merkle_path) in sapling_inputs.into_iter() {
799        let key = match _sapling_key_scope {
800            Scope::External => ufvk.sapling().map(|k| k.fvk().clone()),
801            Scope::Internal => ufvk.sapling().map(|k| k.to_internal_fvk()),
802        };
803
804        builder.add_sapling_spend(
805            key.ok_or(Error::KeyNotAvailable(PoolType::SAPLING))?,
806            sapling_note.clone(),
807            merkle_path,
808        )?;
809    }
810
811    #[cfg(feature = "orchard")]
812    for (orchard_note, merkle_path) in orchard_inputs.into_iter() {
813        builder.add_orchard_spend(
814            ufvk.orchard()
815                .cloned()
816                .ok_or(Error::KeyNotAvailable(PoolType::ORCHARD))?,
817            *orchard_note,
818            merkle_path.into(),
819        )?;
820    }
821
822    #[cfg(feature = "transparent-inputs")]
823    let mut cache = HashMap::<TransparentAddress, TransparentAddressMetadata>::new();
824
825    #[cfg(feature = "transparent-inputs")]
826    let mut metadata_from_address = |addr: TransparentAddress| -> Result<
827        TransparentAddressMetadata,
828        CreateErrT<DbT, InputsErrT, FeeRuleT, ChangeErrT, N>,
829    > {
830        match cache.get(&addr) {
831            Some(result) => Ok(result.clone()),
832            None => {
833                // `wallet_db.get_transparent_address_metadata` includes reserved ephemeral
834                // addresses in its lookup. We don't need to include these in order to be
835                // able to construct ZIP 320 transactions, because in that case the ephemeral
836                // output is represented via a "change" reference to a previous step. However,
837                // we do need them in order to create a transaction from a proposal that
838                // explicitly spends an output from an ephemeral address (only for outputs
839                // already detected by this wallet instance).
840
841                let result = wallet_db
842                    .get_transparent_address_metadata(account_id, &addr)
843                    .map_err(InputSelectorError::DataSource)?
844                    .ok_or(Error::AddressNotRecognized(addr))?;
845                cache.insert(addr, result.clone());
846                Ok(result)
847            }
848        }
849    };
850
851    #[cfg(feature = "transparent-inputs")]
852    let utxos_spent = {
853        let mut utxos_spent: Vec<OutPoint> = vec![];
854        let add_transparent_input = |builder: &mut Builder<_, _>,
855                                     utxos_spent: &mut Vec<_>,
856                                     address_metadata: &TransparentAddressMetadata,
857                                     outpoint: OutPoint,
858                                     txout: TxOut|
859         -> Result<
860            (),
861            CreateErrT<DbT, InputsErrT, FeeRuleT, ChangeErrT, N>,
862        > {
863            let pubkey = ufvk
864                .transparent()
865                .ok_or(Error::KeyNotAvailable(PoolType::Transparent))?
866                .derive_address_pubkey(address_metadata.scope(), address_metadata.address_index())
867                .expect("spending key derivation should not fail");
868
869            utxos_spent.push(outpoint.clone());
870            builder.add_transparent_input(pubkey, outpoint, txout)?;
871
872            Ok(())
873        };
874
875        for utxo in proposal_step.transparent_inputs() {
876            add_transparent_input(
877                &mut builder,
878                &mut utxos_spent,
879                &metadata_from_address(*utxo.recipient_address())?,
880                utxo.outpoint().clone(),
881                utxo.txout().clone(),
882            )?;
883        }
884        for input_ref in proposal_step.prior_step_inputs() {
885            // A referenced transparent step output must exist and be referenced *at most* once.
886            // (Exactly once in the case of ephemeral outputs.)
887            let (address, outpoint) = unused_transparent_outputs
888                .remove(input_ref)
889                .ok_or(Error::Proposal(ProposalError::ReferenceError(*input_ref)))?;
890
891            let address_metadata = metadata_from_address(address)?;
892
893            let txout = &prior_step_results[input_ref.step_index()]
894                .1
895                .build_result
896                .transaction()
897                .transparent_bundle()
898                .ok_or(ProposalError::ReferenceError(*input_ref))?
899                .vout[outpoint.n() as usize];
900
901            add_transparent_input(
902                &mut builder,
903                &mut utxos_spent,
904                &address_metadata,
905                outpoint,
906                txout.clone(),
907            )?;
908        }
909        utxos_spent
910    };
911
912    #[cfg(feature = "orchard")]
913    let orchard_external_ovk = match &ovk_policy {
914        OvkPolicy::Sender => ufvk
915            .orchard()
916            .map(|fvk| fvk.to_ovk(orchard::keys::Scope::External)),
917        OvkPolicy::Custom { orchard, .. } => Some(orchard.clone()),
918        OvkPolicy::Discard => None,
919    };
920
921    #[cfg(feature = "orchard")]
922    let orchard_internal_ovk = || {
923        #[cfg(feature = "transparent-inputs")]
924        if proposal_step.is_shielding() {
925            return ufvk
926                .transparent()
927                .map(|k| orchard::keys::OutgoingViewingKey::from(k.internal_ovk().as_bytes()));
928        }
929
930        ufvk.orchard().map(|k| k.to_ovk(Scope::Internal))
931    };
932
933    // Apply the outgoing viewing key policy.
934    let sapling_external_ovk = match &ovk_policy {
935        OvkPolicy::Sender => ufvk.sapling().map(|k| k.to_ovk(Scope::External)),
936        OvkPolicy::Custom { sapling, .. } => Some(*sapling),
937        OvkPolicy::Discard => None,
938    };
939
940    let sapling_internal_ovk = || {
941        #[cfg(feature = "transparent-inputs")]
942        if proposal_step.is_shielding() {
943            return ufvk
944                .transparent()
945                .map(|k| sapling::keys::OutgoingViewingKey(k.internal_ovk().as_bytes()));
946        }
947
948        ufvk.sapling().map(|k| k.to_ovk(Scope::Internal))
949    };
950
951    #[cfg(feature = "orchard")]
952    let mut orchard_output_meta: Vec<(BuildRecipient<_>, Zatoshis, Option<MemoBytes>)> = vec![];
953    let mut sapling_output_meta: Vec<(BuildRecipient<_>, Zatoshis, Option<MemoBytes>)> = vec![];
954    let mut transparent_output_meta: Vec<(
955        BuildRecipient<_>,
956        TransparentAddress,
957        Zatoshis,
958        StepOutputIndex,
959    )> = vec![];
960
961    for (&payment_index, output_pool) in proposal_step.payment_pools() {
962        let payment = proposal_step
963            .transaction_request()
964            .payments()
965            .get(&payment_index)
966            .expect(
967                "The mapping between payment index and payment is checked in step construction",
968            );
969        let recipient_address = payment.recipient_address();
970
971        let add_sapling_output = |builder: &mut Builder<_, _>,
972                                  sapling_output_meta: &mut Vec<_>,
973                                  to: sapling::PaymentAddress|
974         -> Result<
975            (),
976            CreateErrT<DbT, InputsErrT, FeeRuleT, ChangeErrT, N>,
977        > {
978            let memo = payment.memo().map_or_else(MemoBytes::empty, |m| m.clone());
979            builder.add_sapling_output(sapling_external_ovk, to, payment.amount(), memo.clone())?;
980            sapling_output_meta.push((
981                BuildRecipient::External {
982                    recipient_address: recipient_address.clone(),
983                    output_pool: PoolType::SAPLING,
984                },
985                payment.amount(),
986                Some(memo),
987            ));
988            Ok(())
989        };
990
991        #[cfg(feature = "orchard")]
992        let add_orchard_output =
993            |builder: &mut Builder<_, _>,
994             orchard_output_meta: &mut Vec<_>,
995             to: orchard::Address|
996             -> Result<(), CreateErrT<DbT, InputsErrT, FeeRuleT, ChangeErrT, N>> {
997                let memo = payment.memo().map_or_else(MemoBytes::empty, |m| m.clone());
998                builder.add_orchard_output(
999                    orchard_external_ovk.clone(),
1000                    to,
1001                    payment.amount().into(),
1002                    memo.clone(),
1003                )?;
1004                orchard_output_meta.push((
1005                    BuildRecipient::External {
1006                        recipient_address: recipient_address.clone(),
1007                        output_pool: PoolType::ORCHARD,
1008                    },
1009                    payment.amount(),
1010                    Some(memo),
1011                ));
1012                Ok(())
1013            };
1014
1015        let add_transparent_output =
1016            |builder: &mut Builder<_, _>,
1017             transparent_output_meta: &mut Vec<_>,
1018             to: TransparentAddress|
1019             -> Result<(), CreateErrT<DbT, InputsErrT, FeeRuleT, ChangeErrT, N>> {
1020                // Always reject sending to one of our known ephemeral addresses.
1021                #[cfg(feature = "transparent-inputs")]
1022                if wallet_db
1023                    .find_account_for_ephemeral_address(&to)
1024                    .map_err(Error::DataSource)?
1025                    .is_some()
1026                {
1027                    return Err(Error::PaysEphemeralTransparentAddress(to.encode(params)));
1028                }
1029                if payment.memo().is_some() {
1030                    return Err(Error::MemoForbidden);
1031                }
1032                builder.add_transparent_output(&to, payment.amount())?;
1033                transparent_output_meta.push((
1034                    BuildRecipient::External {
1035                        recipient_address: recipient_address.clone(),
1036                        output_pool: PoolType::TRANSPARENT,
1037                    },
1038                    to,
1039                    payment.amount(),
1040                    StepOutputIndex::Payment(payment_index),
1041                ));
1042                Ok(())
1043            };
1044
1045        match recipient_address
1046            .clone()
1047            .convert_if_network(params.network_type())?
1048        {
1049            Address::Unified(ua) => match output_pool {
1050                #[cfg(not(feature = "orchard"))]
1051                PoolType::Shielded(ShieldedProtocol::Orchard) => {
1052                    return Err(Error::ProposalNotSupported);
1053                }
1054                #[cfg(feature = "orchard")]
1055                PoolType::Shielded(ShieldedProtocol::Orchard) => {
1056                    let to = *ua.orchard().expect("The mapping between payment pool and receiver is checked in step construction");
1057                    add_orchard_output(&mut builder, &mut orchard_output_meta, to)?;
1058                }
1059                PoolType::Shielded(ShieldedProtocol::Sapling) => {
1060                    let to = *ua.sapling().expect("The mapping between payment pool and receiver is checked in step construction");
1061                    add_sapling_output(&mut builder, &mut sapling_output_meta, to)?;
1062                }
1063                PoolType::Transparent => {
1064                    let to = *ua.transparent().expect("The mapping between payment pool and receiver is checked in step construction");
1065                    add_transparent_output(&mut builder, &mut transparent_output_meta, to)?;
1066                }
1067            },
1068            Address::Sapling(to) => {
1069                add_sapling_output(&mut builder, &mut sapling_output_meta, to)?;
1070            }
1071            Address::Transparent(to) => {
1072                add_transparent_output(&mut builder, &mut transparent_output_meta, to)?;
1073            }
1074            #[cfg(not(feature = "transparent-inputs"))]
1075            Address::Tex(_) => {
1076                return Err(Error::ProposalNotSupported);
1077            }
1078            #[cfg(feature = "transparent-inputs")]
1079            Address::Tex(data) => {
1080                if has_shielded_inputs {
1081                    return Err(ProposalError::PaysTexFromShielded.into());
1082                }
1083                let to = TransparentAddress::PublicKeyHash(data);
1084                add_transparent_output(&mut builder, &mut transparent_output_meta, to)?;
1085            }
1086        }
1087    }
1088
1089    for change_value in proposal_step.balance().proposed_change() {
1090        let memo = change_value
1091            .memo()
1092            .map_or_else(MemoBytes::empty, |m| m.clone());
1093        let output_pool = change_value.output_pool();
1094        match output_pool {
1095            PoolType::Shielded(ShieldedProtocol::Sapling) => {
1096                builder.add_sapling_output(
1097                    sapling_internal_ovk(),
1098                    ufvk.sapling()
1099                        .ok_or(Error::KeyNotAvailable(PoolType::SAPLING))?
1100                        .change_address()
1101                        .1,
1102                    change_value.value(),
1103                    memo.clone(),
1104                )?;
1105                sapling_output_meta.push((
1106                    BuildRecipient::InternalAccount {
1107                        receiving_account: account_id,
1108                        external_address: None,
1109                    },
1110                    change_value.value(),
1111                    Some(memo),
1112                ))
1113            }
1114            PoolType::Shielded(ShieldedProtocol::Orchard) => {
1115                #[cfg(not(feature = "orchard"))]
1116                return Err(Error::UnsupportedChangeType(output_pool));
1117
1118                #[cfg(feature = "orchard")]
1119                {
1120                    builder.add_orchard_output(
1121                        orchard_internal_ovk(),
1122                        ufvk.orchard()
1123                            .ok_or(Error::KeyNotAvailable(PoolType::ORCHARD))?
1124                            .address_at(0u32, orchard::keys::Scope::Internal),
1125                        change_value.value().into(),
1126                        memo.clone(),
1127                    )?;
1128                    orchard_output_meta.push((
1129                        BuildRecipient::InternalAccount {
1130                            receiving_account: account_id,
1131                            external_address: None,
1132                        },
1133                        change_value.value(),
1134                        Some(memo),
1135                    ))
1136                }
1137            }
1138            PoolType::Transparent => {
1139                #[cfg(not(feature = "transparent-inputs"))]
1140                return Err(Error::UnsupportedChangeType(output_pool));
1141            }
1142        }
1143    }
1144
1145    // This reserves the ephemeral addresses even if transaction construction fails.
1146    // It is not worth the complexity of being able to unreserve them, because there
1147    // are few failure modes after this point that would allow us to do so.
1148    #[cfg(feature = "transparent-inputs")]
1149    {
1150        let ephemeral_outputs: Vec<(usize, &ChangeValue)> = proposal_step
1151            .balance()
1152            .proposed_change()
1153            .iter()
1154            .enumerate()
1155            .filter(|(_, change_value)| {
1156                change_value.is_ephemeral() && change_value.output_pool() == PoolType::Transparent
1157            })
1158            .collect();
1159
1160        let addresses_and_metadata = wallet_db
1161            .reserve_next_n_ephemeral_addresses(account_id, ephemeral_outputs.len())
1162            .map_err(Error::DataSource)?;
1163        assert_eq!(addresses_and_metadata.len(), ephemeral_outputs.len());
1164
1165        // We don't need the TransparentAddressMetadata here; we can look it up from the data source later.
1166        for ((change_index, change_value), (ephemeral_address, _)) in
1167            ephemeral_outputs.iter().zip(addresses_and_metadata)
1168        {
1169            // This output is ephemeral; we will report an error in `create_proposed_transactions`
1170            // if a later step does not consume it.
1171            builder.add_transparent_output(&ephemeral_address, change_value.value())?;
1172            transparent_output_meta.push((
1173                BuildRecipient::EphemeralTransparent {
1174                    receiving_account: account_id,
1175                    ephemeral_address,
1176                },
1177                ephemeral_address,
1178                change_value.value(),
1179                StepOutputIndex::Change(*change_index),
1180            ))
1181        }
1182    }
1183
1184    Ok(BuildState {
1185        #[cfg(feature = "transparent-inputs")]
1186        step_index,
1187        builder,
1188        #[cfg(feature = "transparent-inputs")]
1189        transparent_input_addresses: cache,
1190        #[cfg(feature = "orchard")]
1191        orchard_output_meta,
1192        sapling_output_meta,
1193        transparent_output_meta,
1194        #[cfg(feature = "transparent-inputs")]
1195        utxos_spent,
1196    })
1197}

1199// `unused_transparent_outputs` maps `StepOutput`s for transparent outputs
1200// that have not been consumed so far, to the corresponding pair of
1201// `TransparentAddress` and `Outpoint`.
1202#[allow(clippy::too_many_arguments)]
1203#[allow(clippy::type_complexity)]
1204fn create_proposed_transaction<DbT, ParamsT, InputsErrT, FeeRuleT, ChangeErrT, N>(
1205    wallet_db: &mut DbT,
1206    params: &ParamsT,
1207    spend_prover: &impl SpendProver,
1208    output_prover: &impl OutputProver,
1209    usk: &UnifiedSpendingKey,
1210    account_id: <DbT as WalletRead>::AccountId,
1211    ovk_policy: OvkPolicy,
1212    fee_rule: &FeeRuleT,
1213    min_target_height: BlockHeight,
1214    prior_step_results: &[(&Step<N>, StepResult<<DbT as WalletRead>::AccountId>)],
1215    proposal_step: &Step<N>,
1216    #[cfg(feature = "transparent-inputs")] unused_transparent_outputs: &mut HashMap<
1217        StepOutput,
1218        (TransparentAddress, OutPoint),
1219    >,
1220) -> Result<
1221    StepResult<<DbT as WalletRead>::AccountId>,
1222    CreateErrT<DbT, InputsErrT, FeeRuleT, ChangeErrT, N>,
1223>
1224where
1225    DbT: WalletWrite + WalletCommitmentTrees,
1226    ParamsT: consensus::Parameters + Clone,
1227    FeeRuleT: FeeRule,
1228{
1229    let build_state = build_proposed_transaction::<_, _, _, FeeRuleT, _, _>(
1230        wallet_db,
1231        params,
1232        &usk.to_unified_full_viewing_key(),
1233        account_id,
1234        ovk_policy,
1235        min_target_height,
1236        prior_step_results,
1237        proposal_step,
1238        #[cfg(feature = "transparent-inputs")]
1239        unused_transparent_outputs,
1240    )?;
1241
1242    // Build the transaction with the specified fee rule
1243    #[cfg_attr(not(feature = "transparent-inputs"), allow(unused_mut))]
1244    let mut transparent_signing_set = TransparentSigningSet::new();
1245    #[cfg(feature = "transparent-inputs")]
1246    for (_, address_metadata) in build_state.transparent_input_addresses {
1247        transparent_signing_set.add_key(
1248            usk.transparent()
1249                .derive_secret_key(address_metadata.scope(), address_metadata.address_index())
1250                .expect("spending key derivation should not fail"),
1251        );
1252    }
1253    let sapling_extsks = &[usk.sapling().clone(), usk.sapling().derive_internal()];
1254    #[cfg(feature = "orchard")]
1255    let orchard_saks = &[usk.orchard().into()];
1256    #[cfg(not(feature = "orchard"))]
1257    let orchard_saks = &[];
1258    let build_result = build_state.builder.build(
1259        &transparent_signing_set,
1260        sapling_extsks,
1261        orchard_saks,
1262        OsRng,
1263        spend_prover,
1264        output_prover,
1265        fee_rule,
1266    )?;
1267
1268    #[cfg(feature = "orchard")]
1269    let orchard_fvk: orchard::keys::FullViewingKey = usk.orchard().into();
1270    #[cfg(feature = "orchard")]
1271    let orchard_internal_ivk = orchard_fvk.to_ivk(orchard::keys::Scope::Internal);
1272    #[cfg(feature = "orchard")]
1273    let orchard_outputs = build_state.orchard_output_meta.into_iter().enumerate().map(
1274        |(i, (recipient, value, memo))| {
1275            let output_index = build_result
1276                .orchard_meta()
1277                .output_action_index(i)
1278                .expect("An action should exist in the transaction for each Orchard output.");
1279
1280            let recipient = recipient.into_recipient_with_note(|| {
1281                build_result
1282                    .transaction()
1283                    .orchard_bundle()
1284                    .and_then(|bundle| {
1285                        bundle
1286                            .decrypt_output_with_key(output_index, &orchard_internal_ivk)
1287                            .map(|(note, _, _)| Note::Orchard(note))
1288                    })
1289                    .expect("Wallet-internal outputs must be decryptable with the wallet's IVK")
1290            });
1291
1292            SentTransactionOutput::from_parts(output_index, recipient, value, memo)
1293        },
1294    );
1295
1296    let sapling_dfvk = usk.sapling().to_diversifiable_full_viewing_key();
1297    let sapling_internal_ivk =
1298        PreparedIncomingViewingKey::new(&sapling_dfvk.to_ivk(Scope::Internal));
1299    let sapling_outputs = build_state.sapling_output_meta.into_iter().enumerate().map(
1300        |(i, (recipient, value, memo))| {
1301            let output_index = build_result
1302                .sapling_meta()
1303                .output_index(i)
1304                .expect("An output should exist in the transaction for each Sapling payment.");
1305
1306            let recipient = recipient.into_recipient_with_note(|| {
1307                build_result
1308                    .transaction()
1309                    .sapling_bundle()
1310                    .and_then(|bundle| {
1311                        try_sapling_note_decryption(
1312                            &sapling_internal_ivk,
1313                            &bundle.shielded_outputs()[output_index],
1314                            zip212_enforcement(params, min_target_height),
1315                        )
1316                        .map(|(note, _, _)| Note::Sapling(note))
1317                    })
1318                    .expect("Wallet-internal outputs must be decryptable with the wallet's IVK")
1319            });
1320
1321            SentTransactionOutput::from_parts(output_index, recipient, value, memo)
1322        },
1323    );
1324
1325    let txid: [u8; 32] = build_result.transaction().txid().into();
1326    assert_eq!(
1327        build_state.transparent_output_meta.len(),
1328        build_result
1329            .transaction()
1330            .transparent_bundle()
1331            .map_or(0, |b| b.vout.len()),
1332    );
1333
1334    #[allow(unused_variables)]
1335    let transparent_outputs = build_state
1336        .transparent_output_meta
1337        .into_iter()
1338        .enumerate()
1339        .map(|(n, (recipient, address, value, step_output_index))| {
1340            // This assumes that transparent outputs are pushed onto `transparent_output_meta`
1341            // with the same indices they have in the transaction's transparent outputs.
1342            // We do not reorder transparent outputs; there is no reason to do so because it
1343            // would not usefully improve privacy.
1344            let outpoint = OutPoint::new(txid, n as u32);
1345
1346            let recipient = recipient.into_recipient_with_outpoint(
1347                #[cfg(feature = "transparent-inputs")]
1348                outpoint.clone(),
1349            );
1350
1351            #[cfg(feature = "transparent-inputs")]
1352            unused_transparent_outputs.insert(
1353                StepOutput::new(build_state.step_index, step_output_index),
1354                (address, outpoint),
1355            );
1356            SentTransactionOutput::from_parts(n, recipient, value, None)
1357        });
1358
1359    let mut outputs: Vec<SentTransactionOutput<_>> = vec![];
1360    #[cfg(feature = "orchard")]
1361    outputs.extend(orchard_outputs);
1362    outputs.extend(sapling_outputs);
1363    outputs.extend(transparent_outputs);
1364
1365    Ok(StepResult {
1366        build_result,
1367        outputs,
1368        fee_amount: proposal_step.balance().fee_required(),
1369        #[cfg(feature = "transparent-inputs")]
1370        utxos_spent: build_state.utxos_spent,
1371    })
1372}

1374/// Constructs a transaction using the inputs supplied by the given proposal.
1375///
1376/// Only single-step proposals are currently supported.
1377///
1378/// Returns a partially-created Zcash transaction (PCZT) that is ready to be authorized.
1379/// You can use the following roles for this:
1380/// - [`pczt::roles::prover::Prover`]
1381/// - [`pczt::roles::signer::Signer`] (if you have local access to the spend authorizing
1382///   keys)
1383/// - [`pczt::roles::combiner::Combiner`] (if you create proofs and apply signatures in
1384///   parallel)
1385///
1386/// Once the PCZT fully authorized, call [`extract_and_store_transaction_from_pczt`] to
1387/// finish transaction creation.
1388#[allow(clippy::too_many_arguments)]
1389#[allow(clippy::type_complexity)]
1390#[cfg(feature = "pczt")]
1391pub fn create_pczt_from_proposal<DbT, ParamsT, InputsErrT, FeeRuleT, ChangeErrT, N>(
1392    wallet_db: &mut DbT,
1393    params: &ParamsT,
1394    account_id: <DbT as WalletRead>::AccountId,
1395    ovk_policy: OvkPolicy,
1396    proposal: &Proposal<FeeRuleT, N>,
1397) -> Result<pczt::Pczt, CreateErrT<DbT, InputsErrT, FeeRuleT, ChangeErrT, N>>
1398where
1399    DbT: WalletWrite + WalletCommitmentTrees,
1400    ParamsT: consensus::Parameters + Clone,
1401    FeeRuleT: FeeRule,
1402    DbT::AccountId: serde::Serialize,
1403{
1404    use std::collections::HashSet;
1405
1406    let account = wallet_db
1407        .get_account(account_id)
1408        .map_err(Error::DataSource)?
1409        .ok_or(Error::AccountIdNotRecognized)?;
1410    let ufvk = account.ufvk().ok_or(Error::AccountCannotSpend)?;
1411    let account_derivation = account.source().key_derivation();
1412
1413    // For now we only support turning single-step proposals into PCZTs.
1414    if proposal.steps().len() > 1 {
1415        return Err(Error::ProposalNotSupported);
1416    }
1417    let fee_rule = proposal.fee_rule();
1418    let min_target_height = proposal.min_target_height();
1419    let prior_step_results = &[];
1420    let proposal_step = proposal.steps().first();
1421    let unused_transparent_outputs = &mut HashMap::new();
1422
1423    let build_state = build_proposed_transaction::<_, _, _, FeeRuleT, _, _>(
1424        wallet_db,
1425        params,
1426        ufvk,
1427        account_id,
1428        ovk_policy,
1429        min_target_height,
1430        prior_step_results,
1431        proposal_step,
1432        #[cfg(feature = "transparent-inputs")]
1433        unused_transparent_outputs,
1434    )?;
1435
1436    // Build the transaction with the specified fee rule
1437    let build_result = build_state.builder.build_for_pczt(OsRng, fee_rule)?;
1438
1439    let created = Creator::build_from_parts(build_result.pczt_parts).ok_or(PcztError::Build)?;
1440
1441    let io_finalized = IoFinalizer::new(created).finalize_io()?;
1442
1443    #[cfg(feature = "orchard")]
1444    let orchard_outputs = build_state
1445        .orchard_output_meta
1446        .into_iter()
1447        .enumerate()
1448        .map(|(i, (recipient, _, _))| {
1449            let output_index = build_result
1450                .orchard_meta
1451                .output_action_index(i)
1452                .expect("An action should exist in the transaction for each Orchard output.");
1453
1454            (output_index, PcztRecipient::from_recipient(recipient))
1455        })
1456        .collect::<HashMap<_, _>>();
1457
1458    #[cfg(feature = "orchard")]
1459    let orchard_spends = (0..)
1460        .map(|i| build_result.orchard_meta.spend_action_index(i))
1461        .take_while(|item| item.is_some())
1462        .flatten()
1463        .collect::<HashSet<_>>();
1464
1465    let sapling_outputs = build_state
1466        .sapling_output_meta
1467        .into_iter()
1468        .enumerate()
1469        .map(|(i, (recipient, _, _))| {
1470            let output_index = build_result
1471                .sapling_meta
1472                .output_index(i)
1473                .expect("An output should exist in the transaction for each Sapling output.");
1474
1475            (output_index, PcztRecipient::from_recipient(recipient))
1476        })
1477        .collect::<HashMap<_, _>>();
1478
1479    let pczt = Updater::new(io_finalized)
1480        .update_global_with(|mut updater| {
1481            updater.set_proprietary(
1482                PROPRIETARY_PROPOSAL_INFO.into(),
1483                postcard::to_allocvec(&ProposalInfo::<DbT::AccountId> {
1484                    from_account: account_id,
1485                    target_height: proposal.min_target_height().into(),
1486                })
1487                .expect("postcard encoding of PCZT proposal metadata should not fail"),
1488            )
1489        })
1490        .update_orchard_with(|mut updater| {
1491            for index in 0..updater.bundle().actions().len() {
1492                updater.update_action_with(index, |mut action_updater| {
1493                    // If the account has a known derivation, add the Orchard key path to the PCZT.
1494                    if let Some(derivation) = account_derivation {
1495                        // orchard_spends will only contain action indices for the real spends, and
1496                        // not the dummy inputs
1497                        if orchard_spends.contains(&index) {
1498                            // All spent notes are from the same account.
1499                            action_updater.set_spend_zip32_derivation(
1500                                orchard::pczt::Zip32Derivation::parse(
1501                                    derivation.seed_fingerprint().to_bytes(),
1502                                    vec![
1503                                        zip32::ChildIndex::hardened(32).index(),
1504                                        zip32::ChildIndex::hardened(
1505                                            params.network_type().coin_type(),
1506                                        )
1507                                        .index(),
1508                                        zip32::ChildIndex::hardened(u32::from(
1509                                            derivation.account_index(),
1510                                        ))
1511                                        .index(),
1512                                    ],
1513                                )
1514                                .expect("valid"),
1515                            );
1516                        }
1517                    }
1518
1519                    if let Some((pczt_recipient, external_address)) = orchard_outputs.get(&index) {
1520                        if let Some(user_address) = external_address {
1521                            action_updater.set_output_user_address(user_address.encode());
1522                        }
1523                        action_updater.set_output_proprietary(
1524                            PROPRIETARY_OUTPUT_INFO.into(),
1525                            postcard::to_allocvec(pczt_recipient).expect(
1526                                "postcard encoding of PCZT recipient metadata should not fail",
1527                            ),
1528                        );
1529                    }
1530
1531                    Ok(())
1532                })?;
1533            }
1534            Ok(())
1535        })?
1536        .update_sapling_with(|mut updater| {
1537            // If the account has a known derivation, add the Sapling key path to the PCZT.
1538            if let Some(derivation) = account_derivation {
1539                let non_dummy_spends = updater
1540                    .bundle()
1541                    .spends()
1542                    .iter()
1543                    .enumerate()
1544                    .filter_map(|(index, spend)| {
1545                        // Dummy spends will already have a proof generation key.
1546                        spend.proof_generation_key().is_none().then_some(index)
1547                    })
1548                    .collect::<Vec<_>>();
1549
1550                for index in non_dummy_spends {
1551                    updater.update_spend_with(index, |mut spend_updater| {
1552                        // All non-dummy spent notes are from the same account.
1553                        spend_updater.set_zip32_derivation(
1554                            sapling::pczt::Zip32Derivation::parse(
1555                                derivation.seed_fingerprint().to_bytes(),
1556                                vec![
1557                                    zip32::ChildIndex::hardened(32).index(),
1558                                    zip32::ChildIndex::hardened(params.network_type().coin_type())
1559                                        .index(),
1560                                    zip32::ChildIndex::hardened(u32::from(
1561                                        derivation.account_index(),
1562                                    ))
1563                                    .index(),
1564                                ],
1565                            )
1566                            .expect("valid"),
1567                        );
1568                        Ok(())
1569                    })?;
1570                }
1571            }
1572
1573            for index in 0..updater.bundle().outputs().len() {
1574                if let Some((pczt_recipient, external_address)) = sapling_outputs.get(&index) {
1575                    updater.update_output_with(index, |mut output_updater| {
1576                        if let Some(user_address) = external_address {
1577                            output_updater.set_user_address(user_address.encode());
1578                        }
1579                        output_updater.set_proprietary(
1580                            PROPRIETARY_OUTPUT_INFO.into(),
1581                            postcard::to_allocvec(pczt_recipient).expect(
1582                                "postcard encoding of PCZT recipient metadata should not fail",
1583                            ),
1584                        );
1585                        Ok(())
1586                    })?;
1587                }
1588            }
1589
1590            Ok(())
1591        })?
1592        .update_transparent_with(|mut updater| {
1593            // If the account has a known derivation, add the transparent key paths to the PCZT.
1594            if let Some(derivation) = account_derivation {
1595                // Match address metadata to the inputs that spend from those addresses.
1596                let inputs_to_update = updater
1597                    .bundle()
1598                    .inputs()
1599                    .iter()
1600                    .enumerate()
1601                    .filter_map(|(index, input)| {
1602                        build_state
1603                            .transparent_input_addresses
1604                            .get(
1605                                &input
1606                                    .script_pubkey()
1607                                    .address()
1608                                    .expect("we created this with a supported transparent address"),
1609                            )
1610                            .map(|address_metadata| {
1611                                (
1612                                    index,
1613                                    address_metadata.scope(),
1614                                    address_metadata.address_index(),
1615                                )
1616                            })
1617                    })
1618                    .collect::<Vec<_>>();
1619
1620                for (index, scope, address_index) in inputs_to_update {
1621                    updater.update_input_with(index, |mut input_updater| {
1622                        let pubkey = ufvk
1623                            .transparent()
1624                            .expect("we derived this successfully in build_proposed_transaction")
1625                            .derive_address_pubkey(scope, address_index)
1626                            .expect("spending key derivation should not fail");
1627
1628                        input_updater.set_bip32_derivation(
1629                            pubkey.serialize(),
1630                            Bip32Derivation::parse(
1631                                derivation.seed_fingerprint().to_bytes(),
1632                                vec![
1633                                    // Transparent uses BIP 44 derivation.
1634                                    44 | ChildNumber::HARDENED_FLAG,
1635                                    params.network_type().coin_type() | ChildNumber::HARDENED_FLAG,
1636                                    u32::from(derivation.account_index())
1637                                        | ChildNumber::HARDENED_FLAG,
1638                                    ChildNumber::from(scope).into(),
1639                                    ChildNumber::from(address_index).into(),
1640                                ],
1641                            )
1642                            .expect("valid"),
1643                        );
1644                        Ok(())
1645                    })?;
1646                }
1647            }
1648
1649            assert_eq!(
1650                build_state.transparent_output_meta.len(),
1651                updater.bundle().outputs().len(),
1652            );
1653            for (index, (recipient, _, _, _)) in
1654                build_state.transparent_output_meta.into_iter().enumerate()
1655            {
1656                updater.update_output_with(index, |mut output_updater| {
1657                    let (pczt_recipient, external_address) =
1658                        PcztRecipient::from_recipient(recipient);
1659                    if let Some(user_address) = external_address {
1660                        output_updater.set_user_address(user_address.encode());
1661                    }
1662                    output_updater.set_proprietary(
1663                        PROPRIETARY_OUTPUT_INFO.into(),
1664                        postcard::to_allocvec(&pczt_recipient)
1665                            .expect("postcard encoding of pczt recipient metadata should not fail"),
1666                    );
1667                    Ok(())
1668                })?;
1669            }
1670
1671            Ok(())
1672        })?
1673        .finish();
1674
1675    Ok(pczt)
1676}

1678/// Finalizes the given PCZT, and persists the transaction to the wallet database.
1679///
1680/// The PCZT should have been created via [`create_pczt_from_proposal`], which adds
1681/// metadata necessary for the wallet backend.
1682///
1683/// Returns the transaction ID for the resulting transaction.
1684#[cfg(feature = "pczt")]
1685pub fn extract_and_store_transaction_from_pczt<DbT, N>(
1686    wallet_db: &mut DbT,
1687    pczt: pczt::Pczt,
1688    spend_vk: &sapling::circuit::SpendVerifyingKey,
1689    output_vk: &sapling::circuit::OutputVerifyingKey,
1690    #[cfg(feature = "orchard")] orchard_vk: &orchard::circuit::VerifyingKey,
1691) -> Result<TxId, ExtractErrT<DbT, N>>
1692where
1693    DbT: WalletWrite + WalletCommitmentTrees,
1694    DbT::AccountId: serde::de::DeserializeOwned,
1695{
1696    use std::collections::BTreeMap;
1697    use zcash_note_encryption::{Domain, ShieldedOutput, ENC_CIPHERTEXT_SIZE};
1698
1699    let finalized = SpendFinalizer::new(pczt).finalize_spends()?;
1700
1701    let proposal_info = finalized
1702        .global()
1703        .proprietary()
1704        .get(PROPRIETARY_PROPOSAL_INFO)
1705        .ok_or_else(|| PcztError::Invalid("PCZT missing proprietary proposal info field".into()))
1706        .and_then(|v| {
1707            postcard::from_bytes::<ProposalInfo<DbT::AccountId>>(v).map_err(|e| {
1708                PcztError::Invalid(format!(
1709                    "Postcard decoding of proprietary proposal info failed: {}",
1710                    e
1711                ))
1712            })
1713        })?;
1714
1715    let orchard_output_info = finalized
1716        .orchard()
1717        .actions()
1718        .iter()
1719        .map(|act| {
1720            let note = || {
1721                let recipient =
1722                    act.output().recipient().as_ref().and_then(|b| {
1723                        ::orchard::Address::from_raw_address_bytes(b).into_option()
1724                    })?;
1725                let value = act
1726                    .output()
1727                    .value()
1728                    .map(orchard::value::NoteValue::from_raw)?;
1729                let rho = orchard::note::Rho::from_bytes(act.spend().nullifier()).into_option()?;
1730                let rseed = act.output().rseed().as_ref().and_then(|rseed| {
1731                    orchard::note::RandomSeed::from_bytes(*rseed, &rho).into_option()
1732                })?;
1733
1734                orchard::Note::from_parts(recipient, value, rho, rseed).into_option()
1735            };
1736
1737            let external_address = act
1738                .output()
1739                .user_address()
1740                .as_deref()
1741                .map(ZcashAddress::try_from_encoded)
1742                .transpose()
1743                .map_err(|e| PcztError::Invalid(format!("Invalid user_address: {}", e)))?;
1744
1745            let pczt_recipient = act
1746                .output()
1747                .proprietary()
1748                .get(PROPRIETARY_OUTPUT_INFO)
1749                .map(|v| postcard::from_bytes::<PcztRecipient<DbT::AccountId>>(v))
1750                .transpose()
1751                .map_err(|e: postcard::Error| {
1752                    PcztError::Invalid(format!(
1753                        "Postcard decoding of proprietary output info failed: {}",
1754                        e
1755                    ))
1756                })?
1757                .map(|pczt_recipient| (pczt_recipient, external_address));
1758
1759            // If the pczt recipient is not present, this is a dummy note; if the note is not
1760            // present, then the PCZT has been pruned to make this output unrecoverable and so we
1761            // also ignore it.
1762            Ok(pczt_recipient.zip(note()))
1763        })
1764        .collect::<Result<Vec<_>, PcztError>>()?;
1765
1766    let sapling_output_info = finalized
1767        .sapling()
1768        .outputs()
1769        .iter()
1770        .map(|out| {
1771            let note = || {
1772                let recipient = out
1773                    .recipient()
1774                    .as_ref()
1775                    .and_then(::sapling::PaymentAddress::from_bytes)?;
1776                let value = out.value().map(::sapling::value::NoteValue::from_raw)?;
1777                let rseed = out
1778                    .rseed()
1779                    .as_ref()
1780                    .cloned()
1781                    .map(::sapling::note::Rseed::AfterZip212)?;
1782
1783                Some(::sapling::Note::from_parts(recipient, value, rseed))
1784            };
1785
1786            let external_address = out
1787                .user_address()
1788                .as_deref()
1789                .map(ZcashAddress::try_from_encoded)
1790                .transpose()
1791                .map_err(|e| PcztError::Invalid(format!("Invalid user_address: {}", e)))?;
1792
1793            let pczt_recipient = out
1794                .proprietary()
1795                .get(PROPRIETARY_OUTPUT_INFO)
1796                .map(|v| postcard::from_bytes::<PcztRecipient<DbT::AccountId>>(v))
1797                .transpose()
1798                .map_err(|e: postcard::Error| {
1799                    PcztError::Invalid(format!(
1800                        "Postcard decoding of proprietary output info failed: {}",
1801                        e
1802                    ))
1803                })?
1804                .map(|pczt_recipient| (pczt_recipient, external_address));
1805
1806            // If the pczt recipient is not present, this is a dummy note; if the note is not
1807            // present, then the PCZT has been pruned to make this output unrecoverable and so we
1808            // also ignore it.
1809            Ok(pczt_recipient.zip(note()))
1810        })
1811        .collect::<Result<Vec<_>, PcztError>>()?;
1812
1813    let transparent_output_info = finalized
1814        .transparent()
1815        .outputs()
1816        .iter()
1817        .map(|out| {
1818            let external_address = out
1819                .user_address()
1820                .as_deref()
1821                .map(ZcashAddress::try_from_encoded)
1822                .transpose()
1823                .map_err(|e| PcztError::Invalid(format!("Invalid user_address: {}", e)))?;
1824
1825            let pczt_recipient = out
1826                .proprietary()
1827                .get(PROPRIETARY_OUTPUT_INFO)
1828                .map(|v| postcard::from_bytes::<PcztRecipient<DbT::AccountId>>(v))
1829                .transpose()
1830                .map_err(|e: postcard::Error| {
1831                    PcztError::Invalid(format!(
1832                        "Postcard decoding of proprietary output info failed: {}",
1833                        e
1834                    ))
1835                })?
1836                .map(|pczt_recipient| (pczt_recipient, external_address));
1837
1838            Ok(pczt_recipient)
1839        })
1840        .collect::<Result<Vec<_>, PcztError>>()?;
1841
1842    let utxos_map = finalized
1843        .transparent()
1844        .inputs()
1845        .iter()
1846        .map(|input| {
1847            ZatBalance::from_u64(*input.value()).map(|value| {
1848                (
1849                    OutPoint::new(*input.prevout_txid(), *input.prevout_index()),
1850                    value,
1851                )
1852            })
1853        })
1854        .collect::<Result<BTreeMap<_, _>, _>>()?;
1855
1856    let transaction = TransactionExtractor::new(finalized)
1857        .with_sapling(spend_vk, output_vk)
1858        .with_orchard(orchard_vk)
1859        .extract()?;
1860    let txid = transaction.txid();
1861
1862    #[allow(clippy::too_many_arguments)]
1863    fn to_sent_transaction_output<
1864        AccountId: Copy,
1865        D: Domain,
1866        O: ShieldedOutput<D, { ENC_CIPHERTEXT_SIZE }>,
1867        DbT: WalletRead + WalletCommitmentTrees,
1868        N,
1869    >(
1870        domain: D,
1871        note: D::Note,
1872        output: &O,
1873        output_pool: ShieldedProtocol,
1874        output_index: usize,
1875        pczt_recipient: PcztRecipient<AccountId>,
1876        external_address: Option<ZcashAddress>,
1877        note_value: impl Fn(&D::Note) -> u64,
1878        memo_bytes: impl Fn(&D::Memo) -> &[u8; 512],
1879        wallet_note: impl Fn(D::Note) -> Note,
1880    ) -> Result<SentTransactionOutput<AccountId>, ExtractErrT<DbT, N>> {
1881        let pk_d = D::get_pk_d(&note);
1882        let esk = D::derive_esk(&note).expect("notes are post-ZIP 212");
1883        let memo = try_output_recovery_with_pkd_esk(&domain, pk_d, esk, output).map(|(_, _, m)| {
1884            MemoBytes::from_bytes(memo_bytes(&m)).expect("Memo is the correct length.")
1885        });
1886
1887        let note_value = Zatoshis::try_from(note_value(&note))?;
1888        let recipient = match (pczt_recipient, external_address) {
1889            (PcztRecipient::External, Some(addr)) => Ok(Recipient::External {
1890                recipient_address: addr,
1891                output_pool: PoolType::Shielded(output_pool),
1892            }),
1893            (PcztRecipient::External, None) => Err(PcztError::Invalid(
1894                "external recipient needs to have its user_address field set".into(),
1895            )),
1896            #[cfg(feature = "transparent-inputs")]
1897            (PcztRecipient::EphemeralTransparent { .. }, _) => Err(PcztError::Invalid(
1898                "shielded output cannot be EphemeralTransparent".into(),
1899            )),
1900            (PcztRecipient::InternalAccount { receiving_account }, external_address) => {
1901                Ok(Recipient::InternalAccount {
1902                    receiving_account,
1903                    external_address,
1904                    note: Box::new(wallet_note(note)),
1905                })
1906            }
1907        }?;
1908
1909        Ok(SentTransactionOutput::from_parts(
1910            output_index,
1911            recipient,
1912            note_value,
1913            memo,
1914        ))
1915    }
1916
1917    #[cfg(feature = "orchard")]
1918    let orchard_outputs = transaction
1919        .orchard_bundle()
1920        .map(|bundle| {
1921            assert_eq!(bundle.actions().len(), orchard_output_info.len());
1922            bundle
1923                .actions()
1924                .iter()
1925                .zip(orchard_output_info)
1926                .enumerate()
1927                .filter_map(|(output_index, (action, output_info))| {
1928                    output_info.map(|((pczt_recipient, external_address), note)| {
1929                        let domain = OrchardDomain::for_action(action);
1930                        to_sent_transaction_output::<_, _, _, DbT, _>(
1931                            domain,
1932                            note,
1933                            action,
1934                            ShieldedProtocol::Orchard,
1935                            output_index,
1936                            pczt_recipient,
1937                            external_address,
1938                            |note| note.value().inner(),
1939                            |memo| memo,
1940                            Note::Orchard,
1941                        )
1942                    })
1943                })
1944                .collect::<Result<Vec<_>, _>>()
1945        })
1946        .transpose()?;
1947
1948    let sapling_outputs = transaction
1949        .sapling_bundle()
1950        .map(|bundle| {
1951            assert_eq!(bundle.shielded_outputs().len(), sapling_output_info.len());
1952            bundle
1953                .shielded_outputs()
1954                .iter()
1955                .zip(sapling_output_info)
1956                .enumerate()
1957                .filter_map(|(output_index, (action, output_info))| {
1958                    output_info.map(|((pczt_recipient, external_address), note)| {
1959                        let domain =
1960                            SaplingDomain::new(sapling::note_encryption::Zip212Enforcement::On);
1961                        to_sent_transaction_output::<_, _, _, DbT, _>(
1962                            domain,
1963                            note,
1964                            action,
1965                            ShieldedProtocol::Sapling,
1966                            output_index,
1967                            pczt_recipient,
1968                            external_address,
1969                            |note| note.value().inner(),
1970                            |memo| memo,
1971                            Note::Sapling,
1972                        )
1973                    })
1974                })
1975                .collect::<Result<Vec<_>, _>>()
1976        })
1977        .transpose()?;
1978
1979    #[allow(unused_variables)]
1980    let transparent_outputs = transaction
1981        .transparent_bundle()
1982        .map(|bundle| {
1983            assert_eq!(bundle.vout.len(), transparent_output_info.len());
1984            bundle
1985                .vout
1986                .iter()
1987                .zip(transparent_output_info)
1988                .enumerate()
1989                .filter_map(|(output_index, (output, output_info))| {
1990                    output_info.map(|(pczt_recipient, external_address)| {
1991                        // This assumes that transparent outputs are pushed onto `transparent_output_meta`
1992                        // with the same indices they have in the transaction's transparent outputs.
1993                        // We do not reorder transparent outputs; there is no reason to do so because it
1994                        // would not usefully improve privacy.
1995                        let outpoint = OutPoint::new(txid.into(), output_index as u32);
1996
1997                        let recipient = match (pczt_recipient, external_address) {
1998                            (PcztRecipient::External, Some(addr)) => {
1999                                Ok(Recipient::External {
2000                                    recipient_address: addr,
2001                                    output_pool: PoolType::Transparent,
2002                                })
2003                            }
2004                            (PcztRecipient::External, None) => Err(PcztError::Invalid(
2005                                "external recipient needs to have its user_address field set".into(),
2006                            )),
2007                            #[cfg(feature = "transparent-inputs")]
2008                            (PcztRecipient::EphemeralTransparent { receiving_account }, _) => output
2009                                .recipient_address()
2010                                .ok_or(PcztError::Invalid(
2011                                    "Ephemeral outputs cannot have a non-standard script_pubkey"
2012                                        .into(),
2013                                ))
2014                                .map(|ephemeral_address| Recipient::EphemeralTransparent {
2015                                    receiving_account,
2016                                    ephemeral_address,
2017                                    outpoint,
2018                                }),
2019                            (
2020                                PcztRecipient::InternalAccount {
2021                                    receiving_account,
2022                                },
2023                                _,
2024                            ) => Err(PcztError::Invalid(
2025                                "Transparent output cannot be InternalAccount".into(),
2026                            )),
2027                        }?;
2028
2029                        Ok(SentTransactionOutput::from_parts(
2030                            output_index,
2031                            recipient,
2032                            output.value,
2033                            None,
2034                        ))
2035                    })
2036                })
2037                .collect::<Result<Vec<_>, ExtractErrT<DbT, _>>>()
2038        })
2039        .transpose()?;
2040
2041    let mut outputs: Vec<SentTransactionOutput<_>> = vec![];
2042    #[cfg(feature = "orchard")]
2043    outputs.extend(orchard_outputs.into_iter().flatten());
2044    outputs.extend(sapling_outputs.into_iter().flatten());
2045    outputs.extend(transparent_outputs.into_iter().flatten());
2046
2047    let fee_amount = Zatoshis::try_from(transaction.fee_paid(|outpoint| {
2048        utxos_map
2049            .get(outpoint)
2050            .copied()
2051            // Error doesn't matter, this can never happen because we constructed the
2052            // UTXOs map and the transaction from the same PCZT.
2053            .ok_or(BalanceError::Overflow)
2054    })?)?;
2055
2056    // We don't need the spent UTXOs to be in transaction order.
2057    let utxos_spent = utxos_map.into_keys().collect::<Vec<_>>();
2058
2059    let created = time::OffsetDateTime::now_utc();
2060
2061    let transactions = vec![SentTransaction::new(
2062        &transaction,
2063        created,
2064        BlockHeight::from_u32(proposal_info.target_height),
2065        proposal_info.from_account,
2066        &outputs,
2067        fee_amount,
2068        #[cfg(feature = "transparent-inputs")]
2069        &utxos_spent,
2070    )];
2071
2072    wallet_db
2073        .store_transactions_to_be_sent(&transactions)
2074        .map_err(Error::DataSource)?;
2075
2076    Ok(txid)
2077}

2079/// Constructs a transaction that consumes available transparent UTXOs belonging to the specified
2080/// secret key, and sends them to the most-preferred receiver of the default internal address for
2081/// the provided Unified Spending Key.
2082///
2083/// This procedure will not attempt to shield transparent funds if the total amount being shielded
2084/// is less than the default fee to send the transaction. Fees will be paid only from the
2085/// transparent UTXOs being consumed.
2086///
2087/// Parameters:
2088/// * `wallet_db`: A read/write reference to the wallet database
2089/// * `params`: Consensus parameters
2090/// * `spend_prover`: The [`sapling::SpendProver`] to use in constructing the shielded
2091///   transaction.
2092/// * `output_prover`: The [`sapling::OutputProver`] to use in constructing the shielded
2093///   transaction.
2094/// * `input_selector`: The [`InputSelector`] to for note selection and change and fee
2095///   determination
2096/// * `usk`: The unified spending key that will be used to detect and spend transparent UTXOs,
2097///   and that will provide the shielded address to which funds will be sent. Funds will be
2098///   shielded to the internal (change) address associated with the most preferred shielded
2099///   receiver corresponding to this account, or if no shielded receiver can be used for this
2100///   account, this function will return an error. This procedure will return an error if the
2101///   USK does not correspond to an account known to the wallet.
2102/// * `from_addrs`: The list of transparent addresses that will be used to filter transaparent
2103///   UTXOs received by the wallet. Only UTXOs received at one of the provided addresses will
2104///   be selected to be shielded.
2105/// * `min_confirmations`: The minimum number of confirmations that a previously
2106///   received note must have in the blockchain in order to be considered for being
2107///   spent. A value of 10 confirmations is recommended and 0-conf transactions are
2108///   not supported.
2109///
2110/// [`sapling::SpendProver`]: sapling::prover::SpendProver
2111/// [`sapling::OutputProver`]: sapling::prover::OutputProver
